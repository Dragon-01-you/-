# PythonAnywhere配置详细说明

根据您提供的PythonAnywhere配置页面截图，以下是各项配置的填写说明：

## 1. 源代码和工作目录

**工作目录**: `/home/dragon01`
- 这是您在PythonAnywhere上的主目录

## 2. 源代码

**源代码路径**: `/home/dragon01`
- 您的项目文件（api_server_simple.py, wsgi.py等）应该放在这里

## 3. WSGI配置

**WSGI配置文件**: `/var/www/dragon01_pythonanywhere_com_wsgi.py`
- 点击这个文件的链接进行编辑，内容应如下：

```python
import sys
import os

# 添加项目目录到Python路径
sys.path.insert(0, os.path.abspath(os.path.dirname(__file__)))

# 从api_server_simple导入handler
from api_server_simple import handler

# 这是PythonAnywhere需要的变量名
application = handler
```

## 4. 虚拟环境配置

**Virtualenv路径**: `/home/dragon01/myenv`
- 请先创建虚拟环境：
  1. 点击顶部导航栏的"Consoles"
  2. 选择"Bash"
  3. 运行: `python3 -m venv myenv`
  4. 然后在此处填入路径

## 5. 静态文件配置

如果您的前端文件（index.html, main.js等）也需要部署，请在此部分添加：

| URL | 目录 |
|-----|------|
| /static/ | /home/dragon01/OKComputer_江西工业智能问答前端 |
| / | /home/dragon01/OKComputer_江西工业智能问答前端 |

## 6. 环境变量

添加以下环境变量：
- `PYTHONPATH`: `/home/dragon01`

## 7. 部署步骤回顾

1. **上传项目文件**：
   - 点击顶部导航栏的"Files"
   - 上传以下关键文件：
     - `api_server_simple.py`
     - `vector_db_utils_simple.py`
     - `wsgi.py`
     - `minimal_requirements.txt`
     - 前端文件夹 `OKComputer_江西工业智能问答前端`

2. **安装依赖**：
   - 在Bash控制台中运行：
     ```bash
     source myenv/bin/activate
     pip install -r minimal_requirements.txt
     ```

3. **配置完成后**：
   - 点击绿色的"Reload dragon01.pythonanywhere.com"按钮
   - 访问 `https://dragon01.pythonanywhere.com` 测试应用
   - 检查 `https://dragon01.pythonanywhere.com/health` 确认后端服务正常运行

如果遇到问题，请查看日志文件进行排查。
