# 江西工业工程职业技术学院智能问答系统设计

## 设计理念

### 视觉风格
- **学校特色**：体现江西工业工程职业技术学院的专业性和现代感
- **色彩方案**：以学校主色调墨绿色（RGB: 0, 92, 161）为基础，辅以蓝色和红色作为强调色
- **字体选择**：使用现代无衬线字体，确保清晰易读
- **整体风格**：简洁、专业、现代化，符合教育机构形象

### 配色方案
- **主色调**：墨绿色 #005ca1（学校标准色）
- **辅助色**：深蓝色 #003d73、浅蓝色 #e8f4f8
- **强调色**：红色 #dc2626（用于重要按钮和提示）
- **中性色**：白色 #ffffff、浅灰色 #f8fafc、深灰色 #374151

### 布局设计
- **顶部区域**：校徽 + 学校名称 + 系统标题
- **主体区域**：聊天记录显示区域（占据大部分空间）
- **底部区域**：输入框 + 发送按钮
- **响应式**：适配桌面端（1200px+）、平板端（768px-1199px）、手机端（<768px）

## 视觉效果

### 使用库和技术
- **Anime.js**：实现平滑的动画过渡效果
- **CSS Grid & Flexbox**：实现响应式布局
- **CSS自定义属性**：便于主题色统一管理
- **原生JavaScript**：实现交互逻辑

### 动画效果
- **消息出现动画**：使用Anime.js实现消息渐入效果
- **输入框聚焦动画**：边框颜色渐变和轻微缩放
- **按钮交互动画**：悬停时的颜色变化和轻微位移
- **滚动动画**：平滑滚动到最新消息

### Hero效果
- **渐变背景**：使用CSS渐变创建现代感背景
- **阴影效果**：为卡片和按钮添加适当的阴影层次
- **响应式图片**：校徽和图标在不同设备上的适配显示

### 交互反馈
- **悬停效果**：按钮和链接的悬停状态变化
- **焦点状态**：输入框和按钮的焦点指示
- **加载状态**：优雅的加载动画和状态提示
- **错误状态**：清晰的错误提示和视觉反馈
