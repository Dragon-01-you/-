# 江西工业工程职业技术学院智能问答网页项目大纲

## 文件结构

### HTML文件
- **index.html** - 主页面文件
  - 包含完整的问答界面结构
  - 响应式布局设计
  - 引入CSS和JavaScript文件

### CSS文件
- **styles.css** - 主要样式文件
  - 基础样式重置和全局样式
  - 响应式布局定义
  - 颜色主题和动画效果
  - 组件样式（头部、聊天记录、输入区域）

### JavaScript文件
- **main.js** - 主要交互逻辑
  - 问答功能实现
  - 异步请求处理
  - 动画效果控制
  - 响应式交互适配

### 资源文件夹 (resources/)
- **school-logo.png** - 学校校徽图片
- **favicon.ico** - 网站图标

## 页面结构

### 头部区域 (Header)
- 学校校徽图标
- 学校名称（中文+英文）
- 系统标题"智能问答系统"

### 主体聊天区域 (Chat Area)
- 聊天记录容器
- 用户消息样式（右侧，蓝色主题）
- AI回答样式（左侧，白色背景）
- 时间戳显示
- 自动滚动到最新消息

### 输入区域 (Input Area)
- 多行文本输入框
- 发送按钮（带图标）
- 输入状态指示
- 响应式布局适配

## 功能模块

### 核心问答功能
1. 用户输入问题
2. 异步发送到后端API
3. 显示加载状态
4. 接收并显示回答
5. 支持连续对话

### 用户体验优化
1. 自动滚动到最新消息
2. 输入框聚焦动画
3. 按钮状态反馈
4. 错误处理机制
5. 响应式设计适配

### 视觉效果
1. 使用Anime.js实现平滑动画
2. 学校主题色彩应用
3. 现代化UI设计风格
4. 适配不同屏幕尺寸
